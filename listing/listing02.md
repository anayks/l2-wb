Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
В случае test создается переменная x, как результат функции
x присваивается 1 и мы выходим из блока.
После этого вызывается отложенный вызов (defer), в котором
вызывается постинкремент.
В результате постинкремента, в функции значение x = 2
Только после этого, значение функции возвращается

В случае anotherTest
Мы создаем переменную x так же внутри блока функции, но не как результат
Далее мы присваиваем значение x = 1
И возвращаем результатом функции значение 1
Уже после возвращения, которое нельзя изменить, вызывается defer func()
В котором меняется значение X, но уже, в данном случае, не меняется значение результата
```
